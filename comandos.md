# Comandos
## Git
| comando                   | funcion                    |
| ------------------------- | -------------------------- |
| git --version             | version git                |
| git init                  | iniciar un git             |
| git add .                 | añade archivos             |
| git commit -m "nombre"    | se crea la primera version |
| git branch -M master      | rama de desarrollo         |
| git remote add origin     | ruta repositorio           |
| git push -u origin master | sube los cambios           |

## Node
| comando | funcion                                          |
| ------- | ------------------------------------------------ |
| node -v | Version node                                     |
| npm -v  | Version paquete instalacion librerias de node.js |

## Terminal

| comando  | funcion                  |
| -------- | ------------------------ |
| cd + tab | ubicarse en carpetas     |
| dir      | Elementos del directorio |
| cls      | Limpiar terminal         |
